# Linux System Administration

This folder contains teaching materials, practical labs, commands, assignments, quizzes, and examination resources for the Linux System Administration course.

## Course Information

- **Instructor:** Pheav Seyha
- **Course Title:** Linux System Administration
- **Level:** Beginner to Intermediate
- **Learning Method:** Theory, demonstration, hands-on practice, assignments, and troubleshooting exercises
- **Main Operating System:** Ubuntu Linux / Ubuntu Server
- **Virtualization Platform:** VirtualBox

## Course Description

This course introduces students to Linux operating systems and essential system administration skills.

Students learn how to install Linux, manage files and directories, create users and groups, configure permissions, install software packages, manage processes and services, configure networking, analyze system logs, automate tasks, and troubleshoot common Linux problems.

## Learning Outcomes

After completing this course, students will be able to:

1. Understand Linux architecture and distributions.
2. Install and configure Ubuntu Linux or Ubuntu Server.
3. Use essential Linux terminal commands.
4. Manage files and directories.
5. Create and manage users and groups.
6. Configure file ownership and permissions.
7. Install, update, and remove software packages.
8. Manage processes and system services.
9. Configure basic Linux networking.
10. Connect securely using SSH.
11. Monitor system resources.
12. Analyze Linux system logs.
13. Automate tasks using Cron.
14. Troubleshoot common Linux system problems.
