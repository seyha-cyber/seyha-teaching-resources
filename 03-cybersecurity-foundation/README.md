# Cybersecurity Foundation

This folder contains teaching materials, practical labs, assignments, quizzes, case studies, and examination resources for the Cybersecurity Foundation course.

## Course Information

- **Instructor:** Pheav Seyha
- **Course Title:** Cybersecurity Foundation
- **Level:** Beginner
- **Learning Method:** Theory, demonstrations, practical labs, case studies, group discussion, and quizzes
- **Target Learners:** Students, beginners, office staff, and non-IT learners interested in cybersecurity
- **Recommended Duration:** 10–15 Hours

## Course Description

This course introduces the fundamental concepts of cybersecurity.

Students learn how cyber threats work, how attackers target users and systems, how to protect devices and accounts, how to recognize phishing and social engineering, and how organizations manage security risks.

The course also introduces important cybersecurity frameworks, security principles, network security basics, operating system security, incident response, and safe online practices.

## Learning Outcomes

After completing this course, students will be able to:

1. Explain the basic meaning of cybersecurity.
2. Understand the CIA Triad.
3. Identify common cyber threats and attacks.
4. Recognize phishing and social engineering attempts.
5. Understand malware and ransomware.
6. Apply strong password and account security practices.
7. Use multi-factor authentication.
8. Understand basic network security concepts.
9. Identify common web and application security risks.
10. Understand cybersecurity risk management.
11. Explain the basic incident response process.
12. Apply safe online behavior.
13. Understand ethical and legal responsibilities in cybersecurity.
14. Use basic cybersecurity tools in a safe lab environment.
15. Report basic security incidents clearly.
