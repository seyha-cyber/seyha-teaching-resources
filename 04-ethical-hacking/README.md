# Ethical Hacking

Practical teaching resources, labs, assignments, quizzes, commands, and exams for the Ethical Hacking course.

---

## Course Overview

This course introduces students to the fundamental concepts, methodologies, tools, and practical techniques used in ethical hacking and penetration testing.

Students will practice cybersecurity assessment activities only in authorized and controlled lab environments.

---

## Course Objectives

By the end of this course, students will be able to:

- Understand ethical hacking concepts and legal responsibilities.
- Explain the penetration testing methodology.
- Perform information gathering and reconnaissance.
- Conduct network scanning and service enumeration.
- Identify common system and web vulnerabilities.
- Use cybersecurity tools in an authorized laboratory.
- Analyze vulnerabilities and security risks.
- Create professional penetration testing reports.
- Recommend appropriate security controls and remediation.

---

## Learning Outcomes

After completing this course, students should be able to:

1. Explain the difference between ethical hackers and malicious hackers.
2. Describe the phases of ethical hacking.
3. Use Linux commands for security assessment.
4. Perform network discovery and port scanning.
5. Enumerate services and operating systems.
6. Identify vulnerabilities using security tools.
7. Conduct controlled exploitation in a lab environment.
8. Analyze web application security weaknesses.
9. Document findings and recommend solutions.
10. Follow ethical, legal, and professional standards.
