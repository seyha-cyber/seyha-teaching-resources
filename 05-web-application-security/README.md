# Web Application Security

Practical teaching resources and hands-on laboratories for learning web application security in authorized environments.

## Course Overview

This course introduces web application architecture, HTTP requests and responses, cookies, sessions, authentication, authorization, and common vulnerabilities based on the OWASP Top 10.

Students will practice using intentionally vulnerable platforms such as DVWA and OWASP Juice Shop, together with Burp Suite Community Edition.

## Learning Outcomes

Students will be able to:

- Explain how web applications communicate using HTTP.
- Analyze requests, responses, headers, cookies, and sessions.
- Configure and use Burp Suite Proxy.
- Identify broken access control weaknesses.
- Explain SQL injection and cross-site scripting risks.
- Evaluate authentication and file upload security.
- Document findings and recommend remediation.
- Follow legal, ethical, and professional testing practices.
