# HTTP Request and Response Analysis

## Objective

Describe the purpose and expected learning outcomes of this practical laboratory.

## Requirements

- Authorized web application security lab
- Browser
- Burp Suite Community Edition
- DVWA or OWASP Juice Shop when applicable
- Evidence folder for screenshots and notes

## Tasks

1. Review the lab scope.
2. Configure the required tools.
3. Complete the authorized testing steps.
4. Record commands, requests, responses, and observations.
5. Save screenshots and evidence.
6. Write findings and remediation recommendations.

## Student Deliverables

- Completed lab answers
- Screenshots or exported evidence
- Short findings summary
- Recommended remediation

## Safety Notice

Perform this exercise only in an authorized and controlled laboratory environment.
