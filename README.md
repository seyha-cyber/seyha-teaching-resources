# Web Application Security

## Course Information
- Instructor: Pheav Seyha
- Academy: Cyber Frontier Academy
- Duration: 15 Hours
- Level: Beginner to Intermediate

## Learning Objectives
Students will understand:
- OWASP Top 10
- HTTP requests and responses
- Burp Suite Proxy and Repeater
- Broken Access Control
- SQL Injection
- Cross-Site Scripting

## Lab Environment
- Kali Linux
- OWASP Juice Shop
- Burp Suite Community Edition
- Docker

## Important Notice
Use these labs only in an authorized classroom environment.
